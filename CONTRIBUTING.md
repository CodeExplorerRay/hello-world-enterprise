# Contributing to HelloWorld Enterprise

## 14-Step Contribution Process

1. Fill out the Contribution Intent Form (CIF-2024)
2. Schedule a 30-minute kickoff meeting
3. Create a feature branch following naming convention: `feature/CONTRIB-<number>`
4. Write unit tests (minimum 100% coverage)
5. Write integration tests
6. Update documentation
7. Get code review from 3 senior engineers
8. Pass security review
9. Get approval from Product Manager
10. Get approval from Engineering Manager
11. Get approval from VP of Engineering
12. Deploy to staging
13. Run performance tests
14. Deploy to production

## Code Style

- Use 4 spaces for indentation
- Write comments for every line
- Follow the 47 ADR guidelines

## Commit Messages

Format: `[TYPE] Subject: Description`

Types: FEATURE, BUGFIX, DOC, REFACTOR

## Pull Request Template

- [ ] Tests pass
- [ ] Documentation updated
- [ ] ADR written if architectural change
- [ ] Security review passed
- [ ] Performance impact assessed
